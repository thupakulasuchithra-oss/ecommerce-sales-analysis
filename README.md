# E-commerce Sales Analysis

##  Project Overview
This project analyzes e-commerce sales data using Python and Power BI to derive business insights.

##  Tools Used
- Python (Pandas, Matplotlib)
- Power BI
- SQL (in progress)

##  Key Insights
- Identified top-performing product categories
- Analyzed regional sales performance
- Observed monthly revenue trends
